一、分屏
Shift + Alt + “+”	                              横向分屏
Shift + Alt + “-“	                              竖向分屏
Ctrl + Shift + w	                              取消分屏/关闭当前标签页
Ctrl + Shift + t	                              打开新标签页
二、光标移动
Alt + 方向键	                              分屏间移动光标
三、光标区大小
Alt + Shift + 方向键	                              调整当前分屏区大小
四、复制粘贴
Ctrl + Shift + c	                               复制
Ctrl + Shift + v	                               粘贴
Shift + Insert	                               粘贴
五、全屏
F11	                                               全屏切换
Ctrl + Win + 上下键	                               切换console控制台的最大化
六、翻页
Ctrl + Shift + PageUp/PageeDown	翻页
七、字号
按住Ctrl + 鼠标滚轮	                                调整窗口显示大小
